Accessibility

Our website will be fully committed to making sure that it is accessible for all possible users, in order to achieve this we have attempted to follow the guidelines as produced by the worldwide web consortium. As an example we have provided text alternatives for all images displayed on the web-pages and captions where appropriate to help screen readers to understand pictures. Furthermore, all styles for the website are contained within a separate CSS file, so that a user could apply their own style rules to the website.
